How bd works

Post describing how the program bd works. It is a convenient way to go back to one of the parent directories when you are deep in the directory structure. This program is a shell script.

Main functions

The code functionality is logically divided into 3 functions :
  1. help_msg : Prints some text to help the user.
  2. usage_error : Prints some text to report that the program has not been used incorrectly and directs the user to the github site for information on how to use the program.
  3. newpwd : This function contains the main functionality of the program.
The program execution starts at around line 56 and uses and if condition to check if condition to call one of usage_error or newpwd.
